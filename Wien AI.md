# Wien AI

"Turn user feedback into dev-ready specs" — a product-intelligence tool for PMs that ingests feedback (support tickets, interviews, reviews, surveys), deduplicates and ranks insights by impact/frequency, generates PRDs and technical specs with user stories and effort estimates, and exports to Cursor / Claude Code (plus Jira). On positioning alone it is a near-literal match to [[The Full Discovery Loop]], in the same mold as [[Sprntly]].

## Company

- **Website:** `wien-eylm.vercel.app` (a default Vercel deploy subdomain, not a custom domain). `[Confirmed]`
- **Status:** `[Inference]` Appears **very early / pre-launch** — the unclaimed `*.vercel.app` URL, absence of any company entity, and lack of independent coverage all point to a nascent project rather than an established product.
- **Funding / team / location:** `[Unverified]` — no founder name, funding, Crunchbase/Tracxn entry, or press coverage surfaced (searched 2026-06-01). Cannot classify as bootstrapped *or* VC-backed; filed under indie provisionally, like [[Sprntly]].
- **Pricing (as listed on the site):** Free (3 analyses), Starter $19/mo (50 analyses/mo), Pro $35/mo (unlimited). `[Unverified]`

> [!WARNING]
> The landing page names **FEMSA** and **Grupo Salinas** as customers. This is **`[Unverified]` and currently uncorroborated** — independent sources show Oxxo/FEMSA using Salesforce AI, not Wien AI, and no source ties either enterprise to this product. Treat the customer logos as unconfirmed marketing claims, not evidence of traction. The site is the **only** source for every Wien AI claim here.

## Loop fit

`[Confirmed]` (as stated positioning) Wien AI's pitch — feedback in → ranked insights → PRD + dev tasks → coding-agent export — covers the discovery loop on paper, alongside [[Sprntly]] and [[Trova]]. But as with those, this is positioning, not demonstrated capability. See [[Market Map and the Open Gap]] for why the full loop remains unproven by any verified shipping product.
